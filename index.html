Quem é esse cara?

Meu nome é Odinir.

Sou desenvolvedor web e UX designer de Curitiba no sul do Brasil.

Eu realmente gosto de resolver problemas e também de tornar as coisas bonitas e fáceis de usar. Não consigo parar de aprender coisas novas; quanto mais, melhor. Também adoro fotografia, hobby que levo desde as boas e velhas câmeras de filme.

O que ele faz?

Eu sou um programador.

Para o front-end costumo trabalhar com Javascript, seja standalone ou incluindo frameworks populares como ReactJS e VueJS. Também deixo a web bonita usando Sass, CSS e, sempre que necessário, algum de seus amigos: Bootstrap, Bulma, etc..

Para back-end também trabalho com Javascript (NodeJS, Express, MongoDB, etc). Mas, claro, sempre que o projeto requer PHP, eu faço PHP também (Wordpress, Laravel, etc).

E tambem sou designer.

Sinto-me confortável trabalhando com muitos produtos Adobe. Photoshop, Illustrator, InDesign, Lightroom ou Xd são alguns tipos de padrões da indústria e adoro trabalhar com eles. Mas não estou limitado a eles: Gimp, Inkscape ou Figma também são aplicativos muito válidos com os quais tenho trabalhado.

Interfaces de usuário, brochuras, livros, branding... Você escolhe! Como mencionei, criar coisas bonitas é particularmente importante para mim.
